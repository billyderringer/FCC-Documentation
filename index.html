HSK-12 API Docs

HSK-12

HSK-12 is a custom API that I built to serve as the backend for an open source homeschool content management system called Homeschool Hub (in development).

Installation

Install HSK-12 API before running Homeschool Hub. MongoDB must be running before launching the server.

$ git clone https://github.com/billyderringer/hsk-12.git $ cd hsk-12 $ npm install Start MongoDB server $ npm run dev

Tech Used

Built with:

  • Node.js - v 8.9.4
  • Express - v 4.16.2
  • MongoDB - v 3.6.3
  • Mongoose - v 4.13.6

Features

You can use this API as a standalone or combined with Homeschool Hub (in development).

Use API

Logic:

  • Create teacher
  • Create school terms in specified teacher
  • Create students in specified term
  • Create subjects in specified student
  • Create assignments in specified subject

Routes:

Base URL: http://localhost:3005/api/v1

/teacher

/register - register new account

/login - returns jwt

/logout

/me - returns teacher ObjectId

/teacherId - get teacher by id

/update/teacherId - update teacher info

/remove/teacherId - delete teacher and all children

/term

/student

/subject

/assignment

/create/parentId - create new element

/parent/elementId - get all elements in parent

/elementId - get element by elementId

/update/elementId - update element by elementId

/remove/elementId - remove element and children by elementId
